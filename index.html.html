What is CTF?

Capture The Flag (CTF) is a cybersecurity competition where participants solve challenges to find hidden "flags". These flags are usually strings of text that prove you've solved the challenge. CTFs help develop real-world security skills in a safe, legal environment.

Common CTF Categories

CTF challenges are typically divided into categories. Each category tests different cybersecurity skills.

Web Exploitation

Find and exploit vulnerabilities in web applications. Learn about SQL injection, XSS, CSRF, and more.

CRYPTO

Caesar Cipher

75 points
Beginner

The Caesar cipher is one of the simplest encryption techniques. Each letter is shifted by a fixed number of positions in the alphabet. Decrypt this message to find the flag.

Encrypted Message:

FDHVDU LV IXQ

Shift: 3 positions

Hint

A Caesar cipher with shift 3 means A becomes D, B becomes E, etc. To decrypt, shift backward by 3.

Solution

Shift each letter back by 3 positions: F→C, D→A, H→E, V→S, D→A, U→R, L→I, V→S, I→F, X→U, Q→N

Decrypted: CAESAR IS FUN
Flag: CTF{caesar_is_fun}

JavaScript Obfuscation

125 points
Intermediate

Sometimes developers obfuscate JavaScript code to hide functionality. Can you deobfuscate this code to find the flag?

Obfuscated JavaScript:

function checkFlag(input) {
    var a = "C";
    var b = "T";
    var c = "F";
    var d = "{";
    var e = "j";
    var f = "s";
    var g = "_";
    var h = "d";
    var i = "e";
    var j = "o";
    var k = "b";
    var l = "f";
    var m = "u";
    var n = "s";
    var o = "c";
    var p = "a";
    var q = "t";
    var r = "e";
    var s = "}";
    
    var flag = a + b + c + d + e + f + g + h + i + j + k + g + l + m + n + o + p + q + r + i + s;
    
    if (input === flag) {
        return true;
    }
    return false;
}

Hint

The code builds the flag by concatenating single-character strings. Try running the code in a JavaScript console or manually concatenate the variables.

Solution

Concatenate all the variables in order:

a + b + c + d + e + f + g + h + i + j + k + g + l + m + n + o + p + q + r + i + s
= "C" + "T" + "F" + "{" + "j" + "s" + "_" + "d" + "e" + "o" + "b" + "_" + "f" + "u" + "s" + "c" + "a" + "t" + "e" + "e" + "}"
= CTF{js_deobfuscatee}
